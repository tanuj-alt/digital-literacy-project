# digital-literacy-project
Overview
This project focuses on understanding cybercrime, online safety, and using digital tools effectively as a college student. It explores scenarios like UPI fraud, phishing, and miscommunication in group projects, highlighting how small mistakes online can have real consequences.

Project Contents
Case Study: A detailed look at UPI/online payment fraud, including how it happens, who is targeted, and potential consequences.
Prevention Checklist: A practical “Stay Safe Online” guide with tips for students, including financial safety and reporting channels like National Cyber Crime Portal
 and helpline 1930.
Reflections: Insights on digital communication, lessons learned from research, and habits to improve online safety.
Digital Tools Used:
Google Docs – Organizing and formatting content
Wikipedia – Researching concepts
HackerRank – Practising coding and problem-solving
Stack Overflow – Seeking coding solutions
LinkedIn – Networking and professional growth
GitHub – Showcasing projects and collaborating
Canva – Designing visually appealing presentations
Learning Outcomes
Improved digital literacy, research, and presentation skills
Better awareness of cyber risks and safe online practices
Enhanced coding, problem-solving, and professional networking skills
How to Use This Project

All files are organized in the task-5-cybercrime folder.

casestudy.md – Case study on UPI fraud
prevention-checklist.md – Online safety checklist
Project reflections – Reflections and learnings
Designs – Canva designs used for visual presentation

This project aims to help students understand online risks, practise safe digital habits, and improve both academic and professional skills.
